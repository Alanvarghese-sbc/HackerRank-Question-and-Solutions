# Python Lists - Operations

## 📌 Problem Statement

Perform operations on a list based on given commands.

Supported commands:
- insert i e
- print
- remove e
- append e
- sort
- pop
- reverse

---

## 🧠 Example

### Input
4
append 1
append 2
insert 1 3
print


### Output


[1, 3, 2]


---

## 🚀 Approach

- Initialize empty list
- Read commands one by one
- Use conditional statements to execute operations
