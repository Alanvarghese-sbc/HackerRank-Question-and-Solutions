# Finding the Percentage

## 📌 Problem Statement

Given the names and marks for a list of students, store them in a dictionary and calculate the average marks of a queried student.

- Input includes student names followed by their marks
- Output should display the average score rounded to 2 decimal places

---

## 🧠 Example

### Input
3
- Krishna 67 68 69
- Arjun 70 98 63
- Malika 52 56 60
- Malika

### Output
56.00

---

## 🚀 Approaches

### 1. Using Built-in Functions

- Use `map()` to convert input to float
- Use `sum()` and `len()` to calculate average
- Use formatted string to print result

---

### 2. Manual Approach

- Convert input manually using loops
- Calculate total using loop instead of `sum()`
- Compute average manually
