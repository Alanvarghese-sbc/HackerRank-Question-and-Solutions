# Tuples - Hash Function

## 📌 Problem Statement

Given integers, create a tuple and print its hash value.

---

## 🧠 Example

Input:
2
1 2

Output:
3713081631934410656

---

## 🚀 Approach

- Convert input to tuple using `tuple(map(int, ...))`
- Use built-in `hash()` function

---

## 💡 Key Concepts

- Tuples (immutable)
- hash() function
