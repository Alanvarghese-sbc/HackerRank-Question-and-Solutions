# String Mutations

## 📌 Problem

Modify a string by replacing a character at a given index.

---

## 🧠 Example

Input:
abracadabra
5 k

Output:
abrackdabra

---

## 🚀 Approaches

### 1. Using Slicing
- Split string into parts
- Replace character
- Join back

### 2. Using List
- Convert string to list
- Modify element
- Convert back to string

---

## 💡 Concepts

- Strings are immutable
- Slicing
- List conversion
